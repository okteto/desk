# Installation

You will need the following components to get started with **Okteto Desk**:

## Okteto Desk

### Hombrew

To install your **Okteto Desk** binary, execute:

```console
brew tap okteto/cli
brew install okteto
```

Upgrade to the latest stable version by executing:

```console
brew upgrade okteto
```

### Manual install

#### Install on MacOS
